Infrastructura Cloud, Facturarea & Prețul FAQ

Q. Ce înseamnă IaaS?

IaaS (Infrastructure as a service - Infrastructură ca Serviciu) este posibilitatea să îți creezi propriul Centru de Date Virtual. Exact ca într-o infrastructură fizică, poți lansa servere, modifica oricând dorești, achita doar pentru ce folosești și crea rețele pentru a le uni împreună.

Q. Server Cloud sau Server Dedicat?

Cel mai mare avantaj a unui Server Cloud este Flexibilitatea. Poți lansa unul în 9 secunde, schimba resursele oricând ai nevoie și achita doar pentru ce ai folosit. Asta înseamnă că poți economisi timpul și banii tăi. Dar un Server Dedicat are și el avantaje, ca de exemplu în cazul când o afacere are cerințe speciale pentru hardware.

Q. Server Cloud sau VPS?

Deobicei companiile oferă VPS-uri ca abonament fix lunar, cu resurse fixe (CPU, RAM, HDD, Trafic). În acest caz nu poți schimba nimic pe parcursul întregii luni. Unele companii oferă VPS-uri achitate pe oră. Însă un Adevărat Provider IaaS oferă posibilitatea de a crea un Centru de Date Virtual, cu Adevărate Funcții de Cloud (Auto-Servire La-Cerere, Routere Virtuale, Firewall-uri Virtuale, Panou de Control Simplu, Acces API etc.). Nu achita doar pentru cuvinte, achită pentru servicii sigure.

Q. Cum sunt facturat pentru Servere Cloud?

Noi facem facturarea pe oră, asta înseamnă că în cazul lansării unui server timp de 3 ore, achiți doar pentru 3 ore. Toate serviciile noastre se achită în avans, deci tu ești în totalitate responsabil pentru menținerea pozitivă și suficientă a Balanței Contului MivoCloud.

Q. Voi fi facturat în timp ce serverul este oprit?

Dacă oprești serverul, nu vei fi facturat pentru CPU și RAM. Achiți doar resursele active: spațiu pe disc, IP-uri, etc.

Q. Oferiți rambursări?

Noi oferim rambursări doar în conformitate cu „Termeni și Condiții”.

Q. Pot să schimb resursele Serverului Cloud?

Da, tu ai posibilitatea de a schimba fiecare resurs aparte (CPU, RAM, HDD, IP) în câteva secunde și o poți face de câte ori ai nevoie. Afacerea ta va fi complet independentă.

Q. Voi avea acces la API și Consolă?

Da, vei avea acces la ambele.

Q. Ce pot face în Panoul de Control MivoCloud?

Este un Panou de Control simplu și intuitiv, unde poți crea, opri, distruge și schimba resursele la un Server Cloud. Deasemenea poți procura IP-uri, crea rețele între serverele tale și pentru plată adițională este disponibil Backup Zilnic.

Q. Oferiți Servicii de Administrare?

Da, noi oferim și servicii de administrare pentru o plată adițională.

Q. Oferiți Backup?

Da, oferim și Backup pentru o plată adițională.

Q. Taxați Traficul de Date?

Nu, vei avea Trafic de Date Nelimitat ( Extern și Intern ).

Q. Ce tip de virtualizare folosi'i?

Noi folosim virtualizarea KVM.
